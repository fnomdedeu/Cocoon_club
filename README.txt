COCOON CLUB MEGEVE — Guide d’utilisation rapide

1. Pour changer une image d’événement :
   - Mets ta nouvelle image dans le dossier “images/”
   - Remplace le fichier correspondant (ex : “dj.jpg” pour la soirée DJ)
   - L’image se mettra à jour toute seule sur le site !

2. Pour ajouter ou retirer des photos de la galerie :
   - Ajoute tes fichiers dans le dossier “images/”
   - Ajoute ou enlève la ligne correspondante dans la section <div class="gallery-images"> du fichier “index.html”
     Exemple :
     <img src="images/photo4.jpg" alt="nouvelle ambiance">
   - Sauvegarde !

3. Mise à jour :
   - Pour toute modification, modifie “index.html”, “style.css” ou le contenu du dossier “images/”
   - Tu peux ensuite recharger ton site (sur GitHub Pages ou autre hébergement).

4. Support :
   - Contact : cocoon_megeve@yahoo.fr ou Facebook CocoonClubMegeve

*Ton site n’affiche plus aucun “+” dans la galerie, ni logo sur les événements, seulement tes vraies photos !*
